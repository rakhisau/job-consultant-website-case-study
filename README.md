Client website case study for a job consultancy business with service pages, candidate inquiry flow, employer inquiry section, responsive design, and live deployment.

# Gayatri International Consultancy Website Case Study

## Overview
This is a client website case study for Gayatri International Consultancy, a job consultancy business. The website was built to create a professional online presence, present recruitment services, and support job seekers and employers through clear inquiry flows.

## Live Website
https://gayatriinternationalconsultancy.com/

## Project Type
Client Website Development

## Business Category
Job Consultancy / Recruitment Services

## Key Features
- Professional homepage design
- Job consultancy service presentation
- Job listings section
- Candidate inquiry flow
- Employer hiring inquiry flow
- Contact section
- Responsive website layout
- Live website deployment

## Tech Stack
- HTML
- CSS
- JavaScript
- Hostinger Horizons
- ChatGPT-assisted development
- Responsive Web Design
- Live Server Deployment

## My Role
I worked on the website structure, page layout, content placement, user flow, responsive design, and deployment support. I used Hostinger Horizons and ChatGPT to build and improve the website for a real client project.
